# Keyboard Shortcuts

Keyboard shortcuts can be enabled in General Settings. Once enabled, pressing
++question++ opens an in-app shortcut reference dialog showing all available
shortcuts.

Shortcuts are disabled when a text input, textarea, or content-editable element
is focused — unless noted otherwise.

## Navigation

Single-letter keys to jump between pages:

| Key    | Page            |
| ------ | --------------- |
| ++h++  | Home            |
| ++c++  | Console         |
| ++p++  | G-code Preview  |
| ++j++  | Jobs            |
| ++i++  | History         |
| ++l++  | Timelapse       |
| ++t++  | Tune            |
| ++g++  | Diagnostics     |
| ++x++  | Configure       |
| ++q++  | System          |
| ++s++  | Settings        |

History, Timelapse, and Diagnostics are only available when those features are
enabled.

## Printing

| Shortcut      | Action       |
| ------------- | ------------ |
| ++shift+p++   | Pause print  |
| ++shift+c++   | Cancel print |

## Tools

| Shortcut      | Action         |
| ------------- | -------------- |
| ++shift+h++   | Home all axes  |

## Actions

| Shortcut              | Action                    |
| --------------------- | ------------------------- |
| ++ctrl+shift+e++      | Emergency stop            |
| ++question++          | Open keyboard shortcuts   |

Emergency stop works everywhere, including inside text inputs and the config
editor.

## File editor

The built-in [file editor](/features/file-editor) has its own set of
shortcuts:

| Shortcut                                  | Action            |
| ----------------------------------------- | ----------------- |
| ++ctrl+s++ / ++cmd+s++                    | Save              |
| ++ctrl+shift+s++ / ++cmd+shift+s++        | Save As           |
| ++ctrl+alt+s++ / ++cmd+alt+s++            | Save and Restart  |
| ++ctrl+shift+e++ / ++cmd+shift+e++        | Emergency Stop    |
| ++ctrl+g++                                | Go to Line        |
| ++ctrl+shift+o++ / ++cmd+shift+o++        | Go to Symbol      |
| ++f1++                                    | Command Palette   |

## Console

| Key                                  | Action                    |
| ------------------------------------ | ------------------------- |
| ++arrow-up++ / ++arrow-down++        | Navigate command history  |
| ++tab++                              | Autocomplete command      |
