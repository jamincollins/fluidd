# Timelapse
{: .no_toc }

---

Fluidd integrates with Moonraker's
[timelapse](https://github.com/mainsail-crew/moonraker-timelapse) component,
providing a dedicated view for managing your timelapse recordings.

From the Timelapse page you can:

- **Browse recordings** — view, download or delete previously captured
  timelapse videos.
- **Monitor progress** — see the current timelapse status during an active
  print.

## Setup

To use this feature you need the Moonraker timelapse component installed and
configured. Refer to the
[moonraker-timelapse documentation](https://github.com/mainsail-crew/moonraker-timelapse)
for installation instructions.
